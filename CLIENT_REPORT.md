# 1. الفكرة بسطر واحد

> **منصة جزائرية تربط من يحتاج خدمة نقل بمن يقدّمها — في الولاية وبين الولايات — مع نظام عمولة شفّاف 10٪.**

أشبه بـ **Uber + Yassir + Lalamove** مدمجة، لكن مُحسَّنة للسوق الجزائري:
- بأسعار يحدّدها السائق (لا تسعير خوارزمي)
- بـ 3 خدمات في تطبيق واحد
- بدفع نقدي حالياً + تحويل CCP/BaridiMob للعمولة (لا حاجة لـ Edahabia merchant agreement)

# 2. السوق

## المشكلة

في الجزائر، طلب النقل اليوم يمر بـ:
- اتصالات هاتفية متعددة لإيجاد سائق
- مفاوضة سعر مع كل واحد منهم
- عدم وجود ضمان أو تتبع
- صعوبة العثور على شاحنة أو moto livraison في الحال

## الحل

**تطبيق واحد، 3 ضغطات، سائق في الطريق.**

| الجمهور | ما يربحه |
|---------|----------|
| **العميل** | يجد سائقاً في دقائق، يرى الأسعار، يقيّم |
| **السائق** | عمل إضافي بدون وسطاء، أجر يصله مباشرة |
| **التجار / المتاجر الإلكترونية** | شبكة توصيل جاهزة بدون توظيف |
| **مالك المنصة (أنت)** | 10٪ من كل عملية، تكاليف تشغيل ~5,000 DA/شهر |

## حجم السوق المستهدف

- 🇩🇿 **58 ولاية، 46 مليون نسمة**
- ~12 مليون مستخدم إنترنت نشط
- ~3 مليون مستخدم Yassir (يثبت قبول السوق)
- الأسطول غير الرسمي (سيارات أجرة + شاحنات صغيرة) > 500,000 سائق محتمل
- لا يوجد منافس مهيمن في **النقل بين الولايات** للبضائع — هذا فراغ كبير

## الـ Stack الكامل

### Frontend
- **Next.js 16.2.2** — أحدث إصدار، يستخدم App Router + React 19
- **React Compiler** مُفعَّل — تحسين أداء تلقائي
- **TypeScript 5 strict** — صفر `any` صامت
- **Tailwind CSS 4** — تصميم متجاوب، dark mode جاهز
- **Lucide React** — مكتبة أيقونات
- **Leaflet** — خرائط مفتوحة المصدر (لا تكلفة Google Maps)

### Backend
- **Next.js API Routes** على Vercel — serverless تتدرّج تلقائياً
- **NextAuth v4** — مصادقة JWT + cookies آمنة
- **Prisma 7** — ORM type-safe
- **PostgreSQL** على Neon (eu-central-1 Frankfurt) — قريب جغرافياً من الجزائر
- **bcryptjs** لـ passwords (10 rounds)
- **Redis (Upstash)** للـ cache + rate limit + pub/sub

### Mobile
- **Capacitor 8** — wrapper Android
- **8 plugins ناتيف**: App, SplashScreen, StatusBar, Network, Haptics, Keyboard, Preferences, **Push Notifications**
- **FCM (Firebase)** للإشعارات
- APK بحجم 85 MB

### Infra (مجاناً للبدء)
- **Vercel Hobby** — استضافة + CDN
- **Neon Free** — قاعدة بيانات
- **Upstash Free** — Redis
- **Vercel Blob** — تخزين صور (1 GB مجاناً)
- **Firebase Cloud Messaging** — مجاني بدون حد
- **Sentry Free** — مراقبة الأخطاء (5k events/شهر)

# 5. الأمان والامتثال القانوني

## أمان البيانات

| الطبقة | المعالجة |
|---------|----------|
| **كلمات السر** | bcrypt 10 rounds — مستحيلة الكسر |
| **الجلسات** | JWT موقّعة + cookies `__Host-` / `__Secure-` |
| **التواصل** | HTTPS/TLS فقط، لا cleartext |
| **SQL Injection** | Prisma queries parameterized 100٪ |
| **CORS** | Whitelist صارم (التطبيق فقط) |
| **CSP** | Frame-ancestors مقيّدة |
| **Admin protection** | لا يمكن حذف admin، لا يمكن لـ admin حذف نفسه |
| **Cascade deletion** | حذف نظيف بدون orphans |

## حماية من الهجمات

| الهجوم | الحماية |
|---------|----------|
| **DDoS تسجيل** | Rate limit 5 تسجيلات/IP/ساعة عبر Redis |
| **Credential stuffing** | 10 محاولات تسجيل دخول/15 دقيقة/معرّف قبل bcrypt |
| **Bot accounts** | تطبيع رقم الهاتف + فحص duplicate + KYC manual |
| **Document fraud** | Admin يراجع كل سائق قبل قبوله |
| **Commission evasion** | حظر تلقائي بعد 7 أيام من التأخر |

## الامتثال للقانون الجزائري

# 6. الأداء والقدرة الاستيعابية

## تحسينات مطبَّقة (v2.0 → v2.6)

| الطبقة | التحسين | الأثر |
|---------|---------|-------|
| **DB indexes** | 14 hot-path index | استعلامات O(log n) حتى مع مليون مستخدم |
| **Pagination** | 50/صفحة على كل قائمة admin | لا يفشل مع 100k مستخدم |
| **Adaptive polling** | 3s نشط، 12s خامل، توقف خلف الخلفية | -70٪ من API calls |
| **Redis cache** | قائمة السائقين (TTL 30s) | -90٪ ضغط DB |
| **Rate limiting** | Redis-based fail-open | يصد bots بدون تأثير على المستخدمين |
| **PrismaNeonHttp** | HTTP بدل WebSocket | cold start ~150ms |
| **React Compiler** | memoization تلقائية | -30٪ re-renders |
| **Static export في APK** | لا SSR، first paint فوري | حتى في offline |
| **Vercel Blob** | الصور خارج DB | DB صغير وسريع |
| **Vercel Cron** | تنظيف يومي للسجلات القديمة | DB لا ينتفخ مع الوقت |

# 7. النموذج التشغيلي

## كيف تكسب المنصة المال

```
عميل يدفع 1,000 DA لطلب توصيل
            │
            ▼
السائق يستلم نقداً من العميل (1,000 DA)
            │
            ▼
في نهاية الشهر، السائق يدين بـ 100 DA (10٪)
            │
            ▼
يحوّل العمولة عبر BaridiMob إلى حسابك CCP
            │
            ▼
Admin يتحقق ويوافق
            │
            ▼
✅ دورة مكتملة
```

## القواعد المُبرمَجة في النظام

- **حد أدنى للدفع:** 500 DA (تجنّب لـ micropayments)
- **موعد التسوية:** نهاية كل شهر
- **مهلة التأخر:** 7 أيام بعد نهاية الشهر
- **العقوبة:** السائق المتأخر يُحظَر تلقائياً من online حتى يدفع
- **معلومات الدفع:** ظاهرة للسائق داخل التطبيق (CCP + اسم الحساب)
- **التحقق:** الـ admin يرى صورة التحويل + رقم العملية + يطابق مع كشف CCP

## مثال أرقام واقعية

### 50 سائق نشط / شهر
- 10 طلبات/سائق متوسط = **500 طلب/شهر**
- متوسط الطلب = 800 DA → إجمالي 400,000 DA
- **عمولتك = 40,000 DA/شهر = ~480,000 DA/سنة**
- تكاليفك = 0 (free tier) → **هامش 100٪**

### 500 سائق نشط / شهر
- 12 طلب/سائق = **6,000 طلب/شهر**
- 6,000 × 800 = 4,800,000 DA
- **عمولتك = 480,000 DA/شهر = ~5,760,000 DA/سنة**
- تكاليفك = ~5,500 DA/شهر = 66,000 DA/سنة → **هامش 98.8٪**

### 5,000 سائق نشط / شهر
- 15 طلب/سائق = **75,000 طلب/شهر**
- 75,000 × 800 = 60,000,000 DA
- **عمولتك = 6,000,000 DA/شهر = ~72,000,000 DA/سنة**
- تكاليفك = ~8,500 DA/شهر → **هامش 99.9٪**

> 💎 النموذج **scalable عالياً** لأن التكاليف ثابتة تقريباً بينما الإيرادات تتضاعف.

# 14. لماذا هذا التطبيق فرصة استثمارية

## 🎯 المزايا التنافسية

1. **بنية تحتية حديثة 2026** (Next.js 16, React 19, Prisma 7) — صعب نسخها
2. **مُحسَّن للسوق الجزائري** بالكامل (CCP/BaridiMob، عربي/فرنسي، 58 ولاية)
3. **3 خدمات في تطبيق واحد** (Uber + Yassir + Lalamove دمج)
4. **نموذج عمولة شفّاف 10٪** — أقل من المنافسين
5. **لا حاجة لـ merchant agreement** بنكي للبدء (cash + CCP)
6. **مجاني للتشغيل في البداية** — لا مخاطرة مالية
7. **scalable** بدون إعادة كتابة معمارية

## 🚀 الفرصة

- السوق الجزائري للنقل: مليارات الدنانير سنوياً
- لا منافس في **النقل بين الولايات للبضائع** (Yassir لا يقدّمها)
- منافسون متخصصون (Yassir = ركاب فقط، Glovo = طعام فقط) — NaqlGo دمج
- ولاء الجمهور للمنتجات الجزائرية (preference for local apps)

## 💼 السيناريوهات

### السنة الأولى (محافظ)
- 200 سائق × 10 طلبات × 800 DA × 10٪ = **160,000 DA/شهر**
- = **1,920,000 DA/سنة** بهامش ~99٪

### السنة الثانية (متوسط)
- 1,000 سائق × 12 طلب × 1,000 DA × 10٪ = **1,200,000 DA/شهر**
- = **14,400,000 DA/سنة**

### السنة الثالثة (طموح)
- 5,000 سائق × 15 طلب × 1,200 DA × 10٪ = **9,000,000 DA/شهر**
- = **108,000,000 DA/سنة**
